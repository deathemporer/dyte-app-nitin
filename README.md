## How To Use

#### Installation



#### Clone repository

```html
    git clone https://github.com/deathemporer/dyte-app-nitin.git
```
#### Move into directory

```html
    cd dyte-app-nitin
```
#### Install node dependencies

```html
    npm install
```
#### Start node server

```html
    npm start
```
